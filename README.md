## ğŸ¯ Usage

### Voice Assistant
1. Select a mode from the horizontal scrolling list
2. Tap the microphone button to start speaking
3. The AI will respond with both text and voice
4. Tap the speaker icon to stop audio playback

### Agentic Designer
1. Click the sparkles icon in the header
2. Choose a tab: Features, Modes, APIs, or MCP
3. Describe what you want to add
4. Click "Generate Enhancement"
5. Review the generated code and activate it

### Custom Modes
1. In the mode selector, click the "+" button
2. Enter mode name and description
3. Your custom mode is now available!
